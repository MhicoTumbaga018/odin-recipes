# Recipes Website

A simple multi-page HTML website featuring three delicious recipes:

* Lumpia
* Fried Rice
* Southern Fried Chicken

This project demonstrates basic HTML concepts such as page structure, links, images, unordered lists, and ordered lists.

## Features

* Homepage with navigation links to each recipe page
* Separate HTML page for each recipe
* Recipe descriptions
* Ingredients listed using unordered lists (`<ul>`)
* Cooking instructions listed using ordered lists (`<ol>`)
* Navigation link back to the homepage from each recipe page


## Built With

* HTML5

## What I Learned

Through this project, I practiced:

* Creating multiple HTML pages
* Using anchor tags (`<a>`) for navigation
* Organizing content with headings and paragraphs
* Creating unordered lists for ingredients
* Creating ordered lists for recipe steps
* Adding and displaying images
* Structuring a simple website using semantic HTML

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.
